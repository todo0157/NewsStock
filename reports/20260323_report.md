## 완료된 작업

### 1. 한국투자증권 API 연동 확인
- `.env.local`에 `KIS_APP_KEY`, `KIS_APP_SECRET` 설정 확인
- `kis-api.ts`의 토큰 발급 → 현재가 조회 → 일별 시세 조회 파이프라인 동작 확인
- 실전투자 URL (`openapi.koreainvestment.com:9443`) 사용

### 2. KOSPI/KOSDAQ 지수 조회 기능 추가
- **`src/lib/stocks/kis-api.ts`** — 지수 관련 함수 추가:
  - `getIndexPrice()`: KOSPI/KOSDAQ 현재 지수, 등락, 등락률, 고가/저가, 거래량 조회
  - `getIndexHistory()`: 지수 일별 시세 조회 (기간별)
  - 지수 코드 매핑: KOSPI=`0001`, KOSDAQ=`2001`, 시장구분코드=`U`
- **`src/app/api/stocks/index/route.ts`** — 지수 API 엔드포인트 신규 생성
  - `GET /api/stocks/index?type=KOSPI&period=1M`
  - `kis-api.ts`의 공유 토큰 캐시 사용 (토큰 충돌 해결)

### 3. 차트 페이지 신규 생성
- **`src/app/(dashboard)/chart/page.tsx`** — 독립 차트 페이지
  - KOSPI/KOSDAQ 지수 카드 (클릭하여 차트 조회)
  - 포트폴리오 종목 버튼 (등록한 종목 선택 가능)
  - Recharts 기반 Area Chart (주가/지수) + Bar Chart (거래량) 토글
  - 기간 선택: 1W / 1M / 3M / 6M / 1Y
  - 시세 상세 정보: 고가, 저가, 거래량, 등락률
  - 상승 시 초록색, 하락 시 빨간색 그라데이션

### 4. 네비게이션 업데이트
- **`src/app/(dashboard)/layout.tsx`** — 사이드바에 "차트" (📈) 메뉴 추가
  - 포트폴리오와 리포트 사이에 위치

### 5. 포트폴리오 페이지 복원
- 이전에 포트폴리오 내부에 넣었던 차트 탭을 제거하고 원래 구조로 복원
- 포트폴리오는 종목 관리(추가/삭제/현재가 조회)에 집중

### 6. recharts 패키지 설치

## 기술적 이슈 및 해결

### 토큰 발급 실패 (403)
- **원인**: `.env.local` 변경 후 dev 서버를 재시작하지 않아 환경변수 미로드
- **해결**: dev 서버 재시작으로 해결

### 토큰 충돌 (유효하지 않은 token)
- **원인**: `index/route.ts`에서 `kis-api.ts`와 별도로 토큰을 발급하여 서로의 토큰을 무효화
- **해결**: `index/route.ts`가 `kis-api.ts`의 공유 토큰/요청 함수를 사용하도록 리팩토링

## 보류 사항

### NXT(넥스트레이드) 대체거래소
- KIS OpenAPI에 NXT 실시간 WebSocket API는 존재하나, REST API(지수/시세 조회)에서 NXT 전용 마켓코드 미확인
- NXT가 2025년 3월 출범 이후 아직 API 지원이 완전하지 않음
- 추후 KIS API에서 공식 지원 시 추가 예정
